# Webpack Starter

This is a boilerplate configuration of webpack that is perfectly suited for the generation of single page applications.

In terms of dependencies and webpack loaders and plugins, it includes:

- webpack
- webpack-cli
- webpack-dev-server
- css-loader
- style-loader
- mini-css-extract-plugin
- html-webpack-plugin
- babel-loader
- @babel/core
- @babel/preset-env
